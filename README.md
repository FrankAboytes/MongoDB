# 🍃 MongoDB App

Aplicación Node.js con MongoDB. CRUD de documentos con Mongoose.

## 🚀 Quick Start

```bash
git clone https://github.com/FrankAboytes/MongoDB.git
cd MongoDB
npm install
# Asegúrate de tener MongoDB corriendo
npm start
```

## 🛠️ Stack

- Node.js + Express
- MongoDB + Mongoose
